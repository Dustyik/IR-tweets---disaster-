API Directory

File Descriptions: This directory contains the server codes for Twitter Information Retrieval project. 
server.py contains the necessary routes for the front end, and hosts the flask server. IR_engine.py is the
file to parse our csv dataset and retrieve article titles and tweets for the frontend. Be sure to start the server
before starting the frontend.

To start the server:
Install the necessary python packages with the command: $pip install -r requirements.txt
Start the server with the command: $python server.py
